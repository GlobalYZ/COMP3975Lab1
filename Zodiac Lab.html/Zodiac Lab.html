The purpose of your application is that, given a year of birth, it will let the user know which zodiac animal he/she belongs to.

Start by creating a folder named zodiac in a suitable working directory.

In your code, add an array named $zodiac that contains the following items:

"Monkey"
"Rooster"
"Dog"
"Pig"
"Rat"
"Ox"
"Tiger"
"Rabbit"
"Dragon"
"Snake"
"Horse"
"Goat"

Use this formula to determine which item in the array to choose, given a year:

$remainder = $year % 12;
zodiac[remainder];
